Button Set Example

Creates multiple buttons without gaps.

Check list of jQueryUI icons

Example 1 $buttons = $page->add('ButtonSet'); $buttons->addButton('Test'); $buttons->addButton('Alert') ->js('click') ->univ() ->alert('BINGOOO!'); $buttons->addButton('With Icon') ->setIcon('ui-icon-star'); Example 2 vertical $buttons = $page->add('ButtonSet',array('vertical'=>true)); $buttons->addButton()->setNoText('ui-icon-star'); $buttons->addButton()->setNoText('ui-icon-trash'); $buttons->addButton()->setNoText('ui-icon-pencil');

Drop buttons introduces a arrow at the end of the button, which is a visual que for a user that it will produce some popover. Anything can be inside that popover.

$button = $page->add('View_Button'); $popover = $form = $button->addPopover(array('width'=>'500')); $popover->add('LoremIpsum',array('paragraphs'=>1,'words'=>50)); $buttons = $popover->add('ButtonSet'); $buttons->addButton('Alert') ->js('click') ->univ() ->alert('BINGOOO!'); $buttons->addButton('Close') ->js('click', $popover->js()->dialog('close'));

While adding rectangular things is fine like that, drop-down menus with submenus do not fit in this well. So there is a separate method:

$button = $page->add('View_Button'); $form = $button->addSplitButton()->addPopover()->add('Form'); $form->addClass('stacked'); $form->addField('Slider','volume'); $button = $page->add('View_Button'); $menu = $button->addMenu(); $menu->addMenuItem('one'); $menu->addMenuItem('two'); $sm = $menu->addSubMenu('submenu'); $menu->addMenuItem('three'); $menu->addMenuItem('four'); $sm -> addMenuItem('half'); $sm -> addMenuItem('quarter'); Example 3 Button with Menu (Split button dropdowns) /* $tb = $page->add('View_ButtonMenuButton')->set('Test'); $menu = $tb->addButtonMenu()->setType('vertical'); $menu->addMenuItem('open','Open...') ->addMenuItem('save','Save') ->sub() ->addMenuItem('saveas','Save as...') ->end() ->addMenuItem('delete','Delete') ; */
